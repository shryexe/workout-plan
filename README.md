# FitLife - Workout Landing Page

A modern, responsive landing page showcasing different workout types built with React, TypeScript, and Tailwind CSS.

## Features

- 9 workout types with detailed descriptions
- Responsive design (mobile-first)
- Dark theme with orange accents
- Smooth animations and hover effects
- Program pricing cards
- Testimonials section

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
